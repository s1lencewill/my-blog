## 关于Hexo

Hexo是一个快速、简洁且高效的博客框架。它基于Node.js，支持Markdown语法，可以快速生成静态网页。

## 主要特性

- **快速生成**: Hexo可以在几秒钟内生成数百个页面
- **Markdown支持**: 使用Markdown编写文章，简单易用
- **主题丰富**: 有大量精美的主题可供选择
- **插件扩展**: 通过插件扩展功能

## 如何使用

### 创建新文章

```bash
hexo new post "文章标题"
```

### 启动本地服务器

```bash
hexo server
```

### 生成静态文件

```bash
hexo generate
```
